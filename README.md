# Android_HW4_JAVA
Aplicación mobile que consume API de peliculas<br/>
La API es https://api.themoviedb.org

<b>Componentes utilizados</b><br/>
Recycler View<br/>
Card View<br/>
Picasso<br/>
Notifications<br/>
Job Scheduler<br/>
Room<br/>
Share Preference<br/>
